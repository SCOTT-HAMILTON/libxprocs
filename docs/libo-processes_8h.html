LIBO_API std::vector<PROCESS> RunningProcesses ( ProcessCondition  callbackCondition,
void *  extraParam 
)

Get the list of running processes. The processes are returned in std::vector. The first argument is a callback function that is invoked with the PROCESS and second argument extraParam if it not NULL everytime a PROCESS is found. If the callback function returns true it is added to the std::vector<PROCESS> to be returned if it returns false the PROCESS is not added to the return value. If the first and seond param is NULL all running process is returned

To get all the processes without any condition or calback the first and second parameters should be NULL

std::vector<PROCESS> processes = RunningProcesses(NULL, NULL);
Parameters
callbackConditionThe call back function that must return either true or false.
extraParamextra parameter passed to the callbackCondition callback function if specified.
Returns
the std::vector of running PROCESSes
